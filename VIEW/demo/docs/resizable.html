## resizable 参数 | 值 | 说明 | ------------ | -------------| -------| tag | `v-resizable` | 标签名 attribute | `css` ,可调整大小组件样式 | 对象表达式写法;`object`类型。 slot | `html` | 放在里面的元素 attribute | `intercept`,当拖动时会调用此方法 | Function类型，形参为x,y; 表示拖动时组件应位于的left，和top值;返回值为{ x: x, y: y } 格式的对象格式; attribute | `handle`,可拖动调整的调整块 | Array类型，可为`n`，`s`， `e`，`w`， `ne`， `nw`， `se`， `sw`其中的一种或多种，s为南，n为北，e为东，w为西。默认为全部。 event | `resze-start`,当开始调整时会调用此方法 | ; event | `resize-move`,当调整中时会调用此方法 | 回调参数为{ w: w, h: h }; event | `resize-stop`,当调整结束时会调用此方法 | ; > resizable组件 提供简单地调整元素大小的操作方式，通常配合 `v-draggable` 组件使用。
